<h2>수치형(Number)</h2>
<h3>숫자 형태로 이루어진 자료형</h3>
<ul>
    <li><h4>정수형(Integer)</h4>
        <ul>
            <li>정수를 표현하는 자료형</li>
            <li>0, 1, 2, 3, ---, -1, -2, -3</li>
        </ul>
    </li>
    <li><h4>실수형(Floating-point)</h4>
        <ul>
            <li>실수를 표현하는 자료형</li>
            <li>1.5, 3.14, 0.0, -567.89, ---</li>
        </ul>
    </li>
    <li><h4>복소수형(Complex)</h4>
        <ul>
            <li>복소수를 표현하는 자료형</li>
        </ul>
    </li>
</ul>
<h2>정수형(Integer)</h2>
<ul>
    <li>정수를 표현하는 자료형(0, 1, 2, 3,---, -1, -2, -3)</li>
</ul>
<h2>실수형(Floating-point)</h2>
<ul>
    <li>실수를 표현하는 자료형(1.5, 3.14, 0.0, -567.89, ---)</li>
    <li>실수의 지수 표현 방식</li>
    <ul>
        <li>1.2X10³과 같이 10의 지수 형태를 표현할 수 있다.</li>
        <li>10의 거듭제곱 부분에 e 또는 E를 기재해고, 그 뒤에 거듭제곱 되는 숫자를 표시한다.(1.2X10³은 1.2e3으로 표기)</li>
    </ul>
</ul>
<h2>문자열(String)</h2>
<ul>
    <li>문자, 단어 등으로 구성된 문자들의 집합</li>
    <li>따옴표로 둘러싸여 있으면 무조건 문자열</li>
    <li>"Hello","안녕","1234"등등</li>
    <li>한 줄로 된 문자열 만들기</li>
    <ul>
        <li>큰 따옴표 ""이나 ''으로 양쪽을 감싼다</li>
    </ul>
    <li>여러 줄로 된 문자열 만들기</li>
    <ul>
        <li>연속된 큰 따옴표 3개 """ """ 으로 양쪽을 감싼다</li>
        <li>연속된 작은 따옴표 3개 ''' ''' 으로 양쪽을 감싼다</li>
    </ul>
    <li>큰 따옴표 또는 작은 따옴표로 문자열을 감싼뒤 \n을 사용한다.</li>
    <li>문자열 안에 따옴표 넣기</li>
        <ul>
            <li>문자열 내용 자체에 넣으려는 따옴표는 다른 종류의 따옴표를 사용해야 한다.</li>
            <li>문자열에 큰 따옴표가 들어가려면 작은 따옴표를 사용해야한다.</li>
        </ul>
    <li>\' 나 \"를 사용하면 따옴표를 사용할수 있다</li>
    <li>탈출 문자(Escape Character)</li>
        <ul>
            <li>\n : 줄바꿈(개행)</li>
            <li>\t : 탭</li>
            <li>\" : 큰 따옴표</li>
            <li>\' : 작은 따옴표</li>
            <li>\\ : 문자 \</li>
        </ul>
</ul>
<h2>논리형(Boolean)</h2>
<ul>
    <li>참과 거짓을 나타내는 자료형</li>
    <li>참은 True, 거짓은 False로 표현한다.</li>
    <li>문자열이 아니기 때문에 각 단어의 양 쪽에 따옴표로 둘러 싸여 있지 않다.</li>
    <li>또한 첫문자는 반드시 대문자로 true,false처럼 소문자로 작성하면 안된다.</li>
    <li><h3>다른 자료를 논리 값으로 변환</h3>
        <ul>
            <li>모든 자료형은 논리형으로 변환될 수 있다.(모든 자료형은 논리 값 참 또는 거짓으로 분류할 수 있다.)</li>
            <li>
                <table border="1">
                    <tr>
                        <th>자료형</th>
                        <th>참</th>
                        <th>거짓</th>
                    </tr>
                    <tr>
                        <td>정수형</td>
                        <td>0이 아닌 정수</td>
                        <td>0</td>
                    </tr>
                    <tr>
                        <td>실수형</td>
                        <td>0.0이 아닌 실수</td>
                        <td>0.0</td>
                    </tr>
                    <tr>
                        <td>문자열</td>
                        <td>1개 이상의 문자가 들어 있는 문자열</td>
                        <td>비어 있는 문자열("")</td>
                    </tr>
                    <tr>
                        <td>논리형</td>
                        <td>True</td>
                        <td>False</td>
                    </tr>
                </table>
            </li>
        </ul>
    </li>
    <li><h3>논리 값을 수치형으로 변환</h3></li>
    <ul>
        <li>논리 값 참과 거짓은 정수 또는 실수로 변환될 수 있다.</li>
        <li>참은 정수 1 또는 실수 1.0으로 간주된다.</li>
        <li>거짓은 정수 0 또는 실수 0.0으로 간주된다.</li>
        <li>
            <table border="1">
                <tr>
                    <th>논리 값</th>
                    <th>정수로 변환할 경우</th>
                    <th>실수로 변환할 경우</th>
                </tr>
                <tr>
                    <td>True</td>
                    <td>1</td>
                    <td>1.0</td>
                </tr>
                <tr>
                    <td>False</td>
                    <td>0</td>
                    <td>0.0</td>
                </tr>
            </table>
        </li>
    </ul>
</ul>
<h2>자료형 확인</h2>
<ul>
    <li>1개의 자료형 확인(ex : type(확인할 1개의 자료) )</li>
</ul>
<h2>자료형 변환</h2>
<ul>
    <li>자료를 정수형으로 변환(ex : int(정수형으로 변환할 1개의 자료) )</li>
    <ul>
        <li>실수형 또는 논리형 자료를 정수형으로 변환할 수 있다.</li>
        <li>문자열은 정수의 형태인 경우 변할 수 있다.</li>
    </ul>
    <li>자료를 실수형으로 변환(ex : float(실수형으로 변환할 1개의 자료) )</li>
    <ul>
        <li>정수형 또는 논리형 자료를 실수형으로 변환할 수 있다.</li>
        <li>문자열은 정수 또는 실수 형태인 경우에 반환할 수 있다.</li>
    </ul>
    <li>자료를 문자열로 변환(ex : str(문자열로 변환할 1개의 자료) )</li>
    <ul>
        <li>모든 기본 자료형들은 문자열로 변환할 수 있다.</li>
    </ul>
    <li>자료를 논리형으로 변환(ex : bool(논리형으로 변환할 1개의 자료) )</li>
    <ul>
        <li>모든 기본 자료형들은 논리형으로 변환할 수 있다.</li>
    </ul>
</ul>
