## 제어(Controller)
+ 프로그램 내의 구문들은 기본적으로 위에서 아래방향으로 순서대로 한번씩 실행된다
+ 상황에 따라 이러한 구문들의 흐름을 제어하여 실행순서를 변경할수 있다

# if
___
### if
+ 조건문을 검사하여 결과가 참이면 그아래 명령어 및 구문들을 순서대로 한번씩 실행한다
+ 검사한 결과가 거짓이면 그아래 구문은 무시하고 실행하지 않는다
+ if조건문 아래에 위치한 구문들은 동일한 간격의 들여쓰기를 해야한다
```python
#사용방법
if:
    True일경우 수행할구문
```
```python
if 3.14>2:
    print("True")
# 실행결과 True
```

### if - else
+ 조건이 참일경우 if 아래 구문을 실행하고 거짓일경우 else아래 구문을 실행한다
+ else 에도 ':'를 붙여야 한다
+ else는 if와 간격이 같아야하며 else밑 구문들도 들여쓰기를 해야한다
```python
if 3.14<2:
    print("False")
else:
    print("True")
```

### elif
+ elif도 else와 같이 if와 간격이 같아야하고 ':'를 붙여야한다
+ if구문이 없이는 사용할수 없다
```python
if 조건1:
    참일경우 실행구문
elif: 조건2
    조건1이 거짓이고 조건2가 참일때 실행구문
elif: 조건3
    조건2가 거짓이고 조건3이 참일때 실행구문
else:
    조건이 모두 거짓일때 실행구문
```
```python
i = int(input("숫자입력:"))
if i>0:
    print("양수")
elif i<0:
    print("음수")
else:
    print("숫자0")
```
### if문에 논리연산자
+ if문에 조건에는 논리연산자를 사용할수 있다
+ "<,>,<=,>=,==,!=","not,and,or"
```python
    num1 = 10
    num2 = -5
    if num1!=num2:
        print("같지않음")
```

### if문의 중접사용
+ 필요함에 따라 if문 안에 if문을 중첩해서 사용할 수 있다
+ 중첩된 하위 블럭은 상위 블럭에 종속된다
```python
num = 100
if num>50:
    if num>100:
        print("num은 100보다 큽니다")
    else:
        print("num은 101보다 작습니다")
```
