# 튜플(Tuple)
+ 리스트와 유사하지만 항목을 변경하는 것이 불가능한 목록 형태의 자료 구조
+ 튜플은 가지고 있는 항목의 값을 수정하거나 삭제할수 없고 추가할수도 없다

### 튜플 만들기
+ 튜플은 여러 값을 쉼표 ','기호로 구분하여 기재하고 값들의 양끝을 소괄호로 씌워서 만들수있다
+ 소괄호는 생략할수 있다
+ 튜플이 가지고 있는 항목이 단 1개인 경우에는 뒤에 ','기호를 기재해야한다
+ 튜플이 가지고 있는 항목이 없는경우에는 쉼표없이 소괄호를 기재한다
```python
    t1 = (1,2,3,4,5)
    t2 = 1,2,3,4,5
    t3 = (1,)
    t4 = ()
    t5 = tuple()
```
### 튜플 내의 1개의 항목 접근
+ 리스트와 동일한 방식으로 1개의 항목을 인덱싱한다
```python
    t1 = (1,2,3,4,5)
    print(t1[0],"===",t1[-1])
```
### 튜플 내의 여러개의 항목 접근
+ 리스트와 동일한 방식으로 항목들을 슬라이싱한다
```python
    t1 = (1,2,3,4,5)
    print(t1[:3],",",t1[-1:2:-1])
```
### 튜플에 대한 연산
```python
    a=(1,2,3)
    b=(1,3,5)
    print("덧셈 = ",a+b)
    print("덧셈2 = ",a+(6,7,8))
    #print(a+b[0]) #Error
    print("덧셈3 = ",a+b[0:1])
    print("곱셈 = ",2*a)
```
### 튜플에 대한 크기 비교
+ 2개의 튜플에서 같은 위치의 항목, 동일한 인덱스의 항목 값을 비교하여 튜플의 대소를 판단한다
+ "==,!="는 다른 자료형과 튜플 간의 비교가 가능
```python
    print((1,2,3,4,5)<(100,))
    print((2,3,4) != "2,3,4")
    print((1,2,3)[-1]==1)
    #print((1,3,5)>=[0,1,2])#Tuple과 List라 Error
```
### 튜플에 특정 값이 존재하는지 확인
+ 찾으려는 값 in 튜플의 이름
+ 찾으려는 값 not in 튜플의 이름
```python
    tuple = (1,2,3)
    print(1 in tuple)
    print(5 not in tuple)
```
### 튜플의 길이 구하기
+ len(튜플의 이름)
+ 대상 튜플에서 항목이 모두 몇 개 존재하는지 확인하여 그개수를 돌려줌
+ 변수에 할당하고 사용할수 있다
```python
    tupleLen = (1,2,3)
    len = len(tupleLen)
    print(len)
```
### 튜플에 있는 항목들의 합계,최대값,최소값 구하기
```python
    ex = (1,2,3,4,5,6,7,8,9)
    sum = sum(ex)
    max = max(ex)
    min = min(ex)
    print("sum = ",sum,", max = ",max,", min = ",min)
```
### 튜플의 자료형 확인하기
+ type를 이용하여 값 또는 변수가 어떤 자료형인지 확인할수 있다
```python
    types = (1,"Hi",3.14,True)
    print("int = ",type(types[0]),", str = ",type(types[1]),", float = ",type(types[2]),", bool = ",type(types[3]),", tuple = ",type(types))
```
### 튜플을 이용한 변수 할당
+ 튜플 자료형을 이용하여 여러 변수 값을 서로 교환하거나 한번에 여러 변수 값을 할당할 수 있다
```python
    ch1 = 1
    ch2 = 3.14
    change1 = (ch1,ch2)
    print(change1)
    ch1,ch2 = ch2,ch1
    print(ch1)
    print(ch2)
```
### 튜플 형태로 나눗셈의 몫과 나머지 구하기
+ divmod는 나눗셈을 수행하여 몫과 나머지를 튜플형태로 돌려준다
```python
    dvm = divmod(7,3)
    print(dvm)#(2,1)
```
### 자료형 변환
+ 리스트 또는 문자열을 튜플 자료형으로 변환. tuple(변환하려는 리스트,문자열)
```python
    a = [1]
    b=tuple(a)
    c = [1,2,["hi","bye"],3,True]
    d=tuple(c)
    print(b)#(1,)
    print(d)#(1,2,['hi','bye'],3,True)
```
+ 튜플 또는 문자열을 리스트 자료형으로 변환. list(변환하려는 튜플,문자열)
```python
    a = (1,)
    b = (1,2,["hi","bye"],True)
    c = list(a)
    d = list(b)
    print(c)#[1]
    print(d)#[1, 2, ['hi', 'bye'], True]
```
