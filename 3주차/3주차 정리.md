<h2>변수(Variables)</h2>
<ul><h4>변수의 개념</h4>
    <li>연산의 대상이 되는 대부분의 값(value)들은 상황에 따라 얼마든지 변할 수 있다</li>
    <li>이렇게 “변하는 값”이 바로 변수이다</li>
    <li>변하는 값을 일관성 있게 관리하기 위해서,그 값에 다른 자료와 구별(식별)할 수 있는 이름(identifier)을 부여한다</li>
    <li>어떠한 값에 이름을 부여하고 나면, 그 때부터 그 값을 이름으로 식별하여 연산에 사용할 수 있다</li>
    <h4>변수 할당(Assignment)</h4>
    <li>어떠한 값에 이름을 부여하는 것을 변수 할당이라고 한다</li>
    <li>변수 할당에는 연산자 기호 ‘=’를 사용한다</li>
    <h4>할당 연산자 축약 표현</h4>
    <li>+,-,*,**,/,//,%은 축약 표현이 가능하다.(ex : i += 1)</li>
    <h4>변수 삭제(del)</h4>
    <li>명령어 del을 사용하면 변수를 제거할수 있음</li>
    <li>변수를 삭제하면 그 변수에 들어 있는 값도 더 이상 유효하지 않는다</li>
</ul>
<h2>변수의 이름을 지을 때 주의할 점</h2>
<ul>
    <li>첫 문자는 알파벳이거나 밑줄 기호로 시작해야함</li>
    <li>두번째 문자부터는 알파벳 또는 밑줄 기호또는 0부터 9까지의 숫자를 사용할수있음</li>
    <li>그외의 특수문자를 사용할수 없다</li>
    <li>대소문자를 구분한다</li>
</ul>
<h4>변수의 이름을 지을 때 의미 상 주의할점</h4>
<ul>
    <li>변수 안에 있는 값이 어떤 의미의 자료인지 사람이 봤을 때 최대한 유추할 수 있도록 작명해야함</li>
</ul>
<h2>주석</h2>
<ul>
    <li>실제로 실행되는 구문이 아니라, 사람이 보고 이해하기위해 기재하는 부분</li>
    <li>기호 '#'뒤에 기재한 모든 내용은 주석으로 간주된다</li>
    <li>ex : #hello</li>
</ul>
