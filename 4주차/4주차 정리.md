# 4주차 정리
# 리스트(List)
___
+ 리스트
    + 말 그대로, 여러 자료들을 목록 형태로 관리하기위한 자료구조
    + 순서가 있는 여러 개의 값을 저장하고 관리
### 리스트 만들기
```python
    # 리스트의 이름 = [항목1,항목2,...]
    index = [1,2,3,4,5]
    indexs = ['a','b','c']
    indexss = [1,2,3,['a','b','c'],4,5]
```
+ 한개의 자료구조 안에 들어있는 각각의 값들을 항목(item)또는 원소/요소(element)라고함
+ 리스트에 있는 항목들은 그위치가 숫자로 매겨져 있는데 0부터 시작한다.
  + 자료 구조에서는 이 항목의 위치에 해당하는 숫자를 index라 한다.
### 리스트 내의 항목 접근
```python
    #리스트의이름[항목의 인덱스]
    index = ['a',2,3.14]
    index[0] #a
    index[1] #2
    index[2] #3.14
```
+ 인덱스는 음수로도 가능하다 맨뒤가 -1로 자료구조의 뒷부분 부터 접근한다
```python
    index = [2,3,1]
    index[-1]#1
    index[-2]#3
    index[-3]#2
```

+ 리스트 내의 1개 항목 접근하는 법
```python
    index = [1,2,3,["Hi","Hello"],4]
    print(index[0])#1
    print(index[3][0])#"Hi"
```

+ 리스트 내의 여러 개의 항목 접근하는 법
  + 여러 개의 항목에 접근하는 것을 슬라이싱(slicing)이라고 한다.
  + 리스트의 이름[시작 인덱스:끝 인덱스:간격]
    + 시작 인덱스 : 생략할경우 기본값 0
    + 끝 인덱스 : 생략할 경우 마지막 위치까지 포함
    + 간격 : 생략할 경우 기본값은 1
  + 역순으로 가지고올수 있음(음수로 지정) 끝인덱스 -1까지 가지고온다.
```python
    index = [1,2,3,4,5]
    print(index[0:4:1])#[1,2,3,4]
    print(index[-2:0:-1])#[4,3,2]
```

### 리스트내에 연산
+ 리스트내에 덧셈연산
```python
    index = [1,2,3]
    indexs = [4,5,6]
    print(index+indexs)#[1,2,3,4,5,6]
    print(index+indexs[0:1])#[1,2,3,4]
```
+ 리스트내에 곱셈연산
```python
    index = [1,2,3]
    print(2*index)#[1,2,3,1,2,3]
```
+ 리스트 크기 비교
  + 2개의 리스트를 비교한다. 같은 항목끼리 비교. 첫번째 항목이 같은 값이아니라면 첫번째걸로 비교후 True,False결정
```python
    index = [1,2,3]
    indexs = [4,5,6]
    indexss = [4,6,5]
    print(index<indexs)#True
    print(indexs>indexss)#False
```
+ 리스트의 특정값 존재 확인
  + in : 찾는 값이 있으면 True반환, 없으면 False반환
  + not in : in과 반대로 찾는 값이 없으면 True, 있으면 False반환
```python
    #in
    index = [1,2,'Hello']
    print(1 in index)#True
    print('Hello' in index)#True
    print('hello' in index)#False

    #not in
    print('Hello' in index)#False
    print('hello' in index)#True
```
### List에 값 추가 삭제 변경
+ List에 항목 추가
```python
    index1 = [1,2,3]
    index1.append(4)
    print(index1[:])#4가 추가됨
    index1.append("Hello")
    index1.append(True)
    print(index1[:])#'Hello'와 True가 추가됨
    #출력결과 : [1,2,3,4,'Hello',True]
```
+ List에 특정위치에 항목 추가
  + listName.insert(추가할 위치, 추가할 값)
  + 추가할 위치는 음수도 가능하다
```python
    index2 = [1,2,3]
    index2.insert(1,4)#2번째 항목에 4추가, 원래 2번째 부터 끝에 있는거 까진 하나씩 밀림
    print(index2[:])#1,4,2,3
```
+ List에 항목 삭제
  + del ListName[indexNum]
    + 슬라이싱으로 한번에 여러개의 항목을 삭제할수도 있다.
  + listName.remove(삭제하려는 항목값)
    + 대상 index에서 첫번째로 나오는 특정 값을 삭제한다.
  + listName.pop(indexNum)
    + 대상 리스트에서 인덱싱한 항목을 가져온다.
    + 인덱스를 생략할수 있고, 이떈 마지막항목을 가지고옴.
    + 값을 지우는것만이 아닌 변수에 할당하고 사용할수 있음.
```python
    #del
    index3 = [1,2,3,4,5]
    del index3[1]
    print(index3[:])#[1,3,4,5]
    del index3[1:]
    print(index3[:])#[1]
    
    #remove
    index4 = [1,2,3,4,5,2]
    index4.remove(2)#정수형2를 삭제한다.
    print(index4[:])#첫번째에 있는 2는 삭제되고 두번째 2는 삭제되지 않는다.

    #pop
    index5 = [1,2,3,4,5]
    deleteindex = index5.pop(2)
    print("index :",index5[:],"deleteindex :",deleteindex)#index : [1,2,4,5] deleteindex : 3
    index5.pop()#생략할경우 마지막 항목이 제거
    print(index5[:])#마지막 5가 사라짐
``` 
### 리스트 정렬, 역순배치, 항목개수 세기, 길이 구하기
  + List의 항목들 정렬
    + ListName.sort()
      + 오름차순 정렬,크기비교가 가능한 자료형들로만 구성되어야동작함
  + List의 항목순서 뒤집기
    + ListName.reverse()
      + 역순으로 배치,정렬을 하는것이 아닌 순서만 역순으로 배치함. sort()를 사용후 reverse()를 사용하면 정렬후 역순으로 배치된다.
  + List의 항목 개수 세기
    + ListName.count(개수를 세려는 항목값)
      + 특정 항목이 몇개 존재하는지 확인 하여 개수를 돌려줌, 변수에 할당해서 사용할수 있음
  + List의 길이 구하기
    + len(ListName)
      + 항목의 개수를 돌려줌, 변수에 할당해서 사용 가능
```python
    #sort()
    index1 = [5,4,87,1,3,4,9]
    print(index1[:])#[5,4,87,1,3,4,9]
    index1.sort()
    print(index1[:])#[1,3,4,4,5,9,87]
    #reverse()
    index1.reverse()
    print(index1[:])#[87,9,5,4,4,3,1]
    #count()
    findCountList = index1.count(4)
    print(findCountList)#2 <- 4가 2개있기 때문
    findCountZero = index1.count(100)
    print(findCountZero)#0 <- 값이 없기 때문
    #len
    lenIndex = [1,2,3,4,5,[1,2,3]]
    nullIndex = []
    print(len(lenIndex))#6
    print(len(nullIndex))#0
```
### List에 있는 항목들의 합계, 최대값, 최소값, 자료형 , 결합
  + List에 합계 구하기
    + sum(ListName)
    + 대상 리스트에 들어가 있는 항목들의 합을 구함, 변수에 할당 가능
    + 덧셈 연산이 가능한 항목들로만 구성되어 있어야함
  + List에 최대값 구하기
    + max(ListName)
    + 항목중 최대값을 구한다, 변수에 할당 가능
    + 서로 크기비교가 가능한 항목들로만 구성되어야 함
  + List에 최소값 구하기
    + min(ListName)
    + max와는 반대
  + List의 자료형 확인하기
    + type(확인할 1개의 자료)
      + 해당 값이 어떤 자료형인지 확인 할수 있다.
  + List에 다른 List 결합
    + ListName.extend(결합할 1개의 리스트)
      + 대상 List의 끝에 다른 List를 결합해 확장함  
      + 리스트에 대한 덧셈 연산과 동일한 기능
```python
    #sum()
    index = [1,2,3]
    sum = sum(index)
    print(sum)#[1,2,3] = 6
    
    #max()
    index = [1,2,3]
    #strNum = [1,2,3,"A","B"]
    max = max(index)
    print(max)
    #Error

    #min()
    index = [1,2,3]    
    min = min(index)

    #type()
    typeIndex = [1,3.14,'Hi',True]
    list = type(typeIndex)
    num = type(typeIndex[0])
    fnum = type(typeIndex[1])
    str = type(typeIndex[2])
    boolNum = type(typeIndex[3])
    print("list :",list,"int :",num,"float :",fnum,"String :",str,"bool :",boolNum)

    #extend()
    exList1 = [1,2,3]
    exList2 = [4,5,6]
    exList1.extend(exList2)
    print(exList1)#[1,2,3,4,5,6]
```
